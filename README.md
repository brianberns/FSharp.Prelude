# FSharp.Prelude

This library is a collection of some useful extensions, aliases, and functions for F#.

Here's an overview of what's inside:

- Error handling
    - AsyncOption
    - AsyncResult
    - AsyncResultOption


- Extensions
    - Async
    - List
    - Option
    - Result
    - String


- FDI (Functional Dependency Injection)
    - Reader
    - ReaderAsyncResult
    - ReaderResult
